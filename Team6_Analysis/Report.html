Short Paragraph

The main accounting procedure, including writing EFT data, processing member data, provider data, and summary report. The data will be stored in a database in the ChocAn Data Center. The hardware needed will be a network infrastructure capable of handling large amounts of data as well as user interface terminals that users can interact with. The software will be used by Acme Accounting Services for billing, and providers will have access to a directory in which they can bill and verify members. Users will interact with the system by logging on and accessing the terminal. The report was written in HTML and submitted by Davis Moman.
